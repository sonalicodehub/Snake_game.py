# Snake Game  

A classic Snake Game built using Python and Pygame, where players control a growing snake to collect food while avoiding collisions with walls and its own body. The game features smooth keyboard controls, score progression through snake growth, and a game-over screen with replay functionality.

## Features

* 🎮 Arrow key controls for snake movement
* 🍎 Randomly generated food items
* 📈 Snake grows longer after eating food
* 💥 Collision detection with walls and snake body
* 🔄 Play Again and Quit options after Game Over
* ⚡ Smooth gameplay using Pygame's game loop and frame rate control

## Technologies Used

* Python 3
* Pygame

## How to Run

1. Install Pygame:

   ```bash
   pip install pygame
   ```
2. Run the Python file:

   ```bash
   python snake_game.py
   ```

## Controls

* ⬅️ Left Arrow – Move Left
* ➡️ Right Arrow – Move Right
* ⬆️ Up Arrow – Move Up
* ⬇️ Down Arrow – Move Down
* C – Play Again (after Game Over)
* Q – Quit Game

## Learning Outcomes

This project demonstrates:

* Game development fundamentals with Pygame
* Event handling and keyboard input
* Collision detection
* Random object generation
* Game loop management
* Basic object movement and rendering
